# 照明シミュレータ開発計画（11/19〜1/20）

## 体制
- **たべちゃん担当**：Unity・DB
- **Bさん担当**：Express・React
- **稼働時間**：1人あたり週5時間（合計10時間/週）
- **完了目標**：2025/1/20（動作するMVP完成）

## 開発ゴール（MVP）
- React：UI実装、画面遷移、Unity埋め込み
- Express：認証／プロジェクト／シーン／ライト設定API
- Unity：ライトの制御（複数）、WebGLビルド、スクショ出力
- DB：必要テーブル（users, projects, scenes, light_states, comments など）
- 主な機能:
  - ログイン  
  - プロジェクト一覧 / 詳細  
  - Unityシミュレータ（ライト操作）  
  - ライト状態の保存・読み込み  
  - シーン一覧（キューシート画面）  
  - コメント  
  - 通知（最低限表示）  
  - ライブラリ（表示のみ）
